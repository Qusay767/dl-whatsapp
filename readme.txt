Replace WooCommerce checkout with WhatsApp ordering system. Modern floating cart with analytics and notifications.

== Description ==

Transform your WooCommerce store with a modern WhatsApp-based checkout system. Perfect for businesses that want to provide personalized customer service and streamline the ordering process.

= Key Features =

* **WhatsApp Integration** - Send orders directly via WhatsApp
* **Floating Cart** - Modern sliding cart with real-time updates
* **Analytics Dashboard** - Track conversions and cart abandonment
* **Smart Notifications** - Real-time user feedback system
* **Mobile Optimized** - Perfect experience on all devices
* **Minimum Order Validation** - Set minimum order amounts
* **Business Hours Display** - Show your operating hours
* **Customizable Colors** - Match your brand colors
* **Safe Mode** - Load only on specific pages with shortcode

= How It Works =

1. Customers add products to their cart
2. They fill out a simple form with their details
3. Order is sent directly to your WhatsApp
4. You can provide personalized service and confirm the order

= Shortcode =

Use `[dlwa_checkout_whatsapp]` on any page or post to display the checkout form.

= Analytics =

Track important metrics:
* Total orders sent
* Cart abandonment rates
* Conversion percentages
* Session-based analytics

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/dl-whatsapp-replace-checkout/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to 'DL WA Checkout' in your admin menu to configure settings
4. Add your WhatsApp number in E.164 format (e.g., +1234567890)
5. Use the shortcode `[dlwa_checkout_whatsapp]` on your checkout page

== Frequently Asked Questions ==

= Do I need WooCommerce? =

Yes, this plugin requires WooCommerce to be installed and active.

= What WhatsApp number format should I use? =

Use the international E.164 format, starting with + followed by country code and number (e.g., +1234567890).

= Can I customize the colors? =

Yes, you can customize both the WhatsApp button color and floating cart button color in the settings.

= Does it work on mobile? =

Yes, the plugin is fully responsive and optimized for mobile devices.

= Can I track analytics? =

Yes, the plugin includes a built-in analytics system to track conversions and cart abandonment.
